# krafton_mini_project

📌깃 사용 시 유의 사항!

```js

터미널 실행 : 윈도우 단축키
    "`+shift+ctrl"
내 상황 체크 (수정한 거 있는 지 , master branch와 동기화 되었는지)

    git status

master branch와 동기화 시키기 !!! 필수!!!!

    git pull origin master

내가 수정한 거 스테이징하기 => git add 으로

     git add . => 모든 변경사항 스테이징

     git add 수정한파일명 => 특정 변경 파일 스테이징하기

커밋 메세지 남기기 !!! 필수!!!!

    git commit -m "커밋메시지입니다"

내 커밋 master branch에 반영하기

    git push origin master

지금까지 있었던 git 기록들 보기

    git log --oneline --graph

* 그외에 branch 만들기..등은 프로젝트 규모가 커지면 + 배워서 써보시길!


```
